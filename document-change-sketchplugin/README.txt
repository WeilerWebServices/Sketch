# Document change sample plugin

Sample plugin demoing the `onDocumentChange` handler.

## Installation

- Download the [latest release](https://github.com/BohemianCoding/document-change-sketchplugin/releases/latest) of the plugin
- Double-click the .zip archive to extract the plugin
- Double-click `document-change.sketchplugin` to install the plugin

## Usage

1. Open a new or existing document
1. Make any change to the document, e.g.
   - Add, delete or rearrange a layer within the _Layers_ sidebar
   - Move or rotate a layer on the canvas
   - Change the fill or border style of a layer
1. Any change will result in a message being displayed at the bottom of the
   document window with details about the change
